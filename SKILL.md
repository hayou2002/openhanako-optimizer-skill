## 三大板块扫描内容

### 工作文件夹（覆盖板块 ①②⑤）

**第一步：预归类**（文件按类型归位）
扫描工作区根目录（`_temp/_tools/data/projects/个性化/` 之外的所有文件），
按【归类映射表】匹配目标目录 → 列出建议去向 → 逐批确认（≤5项/批）→ 执行移动。
详见附录B。

**第二步：规则项扫描**
- ① 工作区：根目录越界、项目待归位、`_temp/` 积压
- ② 缓存：日志（保留3天）、检查点（保留10个）、图片缓存（7天前）
- ⑤ 会话：空目录清理、重复文件检测、大文件预警（>50MB）
- `_temp/` 缓冲区：截图>7天 / 数据>14天 / 脚本>14天 / 导出>30天 / 一次性和工作区残留>7天 / 项目结项≥14天→提示移入 projects/
- `data/`：备份>7份确认、归档激活提示
- `_tools/`：闲置脚本>90天提示归档

### 插件技能（覆盖板块 ③⑥⑧）

- ③ 技能：同名/相似(>70%)/废弃(bundle一致)/工作区副本
- ⑥ 插件：孤儿目录/token-tracker>5MB/单目录>50MB/patcher兼容性
- ⑧ 开发：未结项项目/编译残留

### 系统运行（覆盖板块 ④⑦⑨）

- ④ 经验与记忆：pinned超量(>12~15条)/经验库合并(同类>10条)/经验提炼(experience/→Hanako/技能/)/经验库格式同步/认知沉淀/辅助提炼(MemOS提醒)
- ⑦ 配置：设置合理性审计（只读）
- ⑨ 运行时：session积压、`.ephemeral/>7天`、`uploads/temp-upload`陈旧文件、缓存总容量>500MB、plugin-backups/>30天、旧备份整理

## 核心执行流程

> 选一个板块，走四步。支持连贯操作模式：执行完后询问"还要处理其他吗？直接说下一个就行"，菜单仅在你明确需要时重新展示。

### 阶段1：扫描 → 阶段2：报告 → 阶段3：确认执行 → 阶段4：沉淀

### 确认交互规则

1. 每次最多列出 5 项待处理项，你逐一回复「做」或「跳过」
2. 可回复「全部做」批量接受剩余项
3. 低风险操作（移动.md、删除临时文件）汇总执行，不逐条确认
4. 中风险操作（移动数据文件、归档项目）给出分组建议批量确认
5. 高风险操作（删除.exe、卸载插件）才逐条确认
6. 随时说「中断」停止当前操作

### 留痕与异常处理

- 操作留痕保存到 `_temp/检查点/{日期}-{板块}.md`，保留最新 10 份
- 留痕格式：

```markdown
# 2026-07-12 工作文件夹检查点

| 操作时间 | 项目 | 结果 | 备注 |
|----------|------|------|------|
| 12:54 | 归类 笔记.md → _tools/参考/ | ✅ 已移动 | |
| 12:55 | 删除 Thumbs.db | ✅ 已删除 | |
```

- 扫描结果为空时反馈："工作区很干净，没有需要处理的项目 ⭐"
- 你说错了词 → 提示："当前可选：工作文件夹 / 插件技能 / 系统运行 / 初始化 / 退出"
- 「全部」模式中途想中断 → 说「中断」随时停止

## 附录A：初始化参考文档内容

### 笔记排版语法规则

`**加粗**` = 概念名、结构标签
`==高亮==` = 只标核心关键词，不标整句
禁用 HTML标签（span/u等会显示为代码）
标题层级：一、→（一）→ 1. →（1）
符号约定：⚠️ 易混 | 🔤 口诀 | 💡 理解关键 | 📝 题型 | ⭐ 考情
3条以上对比用表格；标题→考试信息行→内容（固定顺序）；按老师讲解逻辑组织

### 重点实战笔记规则

每科用逻辑链串讲→拆成3-7个组块（每块2-4点）→识别词加粗→一行简记→🔤口诀💡理解注📝考题形式→❌✅易错对比。
融合组块化/双重编码/精细加工记忆原理。

## 附录B：归类映射与判断规则

### 启发式判断规则

优先匹配文件名关键字，无匹配则询问用户：

| 关键字（包含任意一个） | 归类 |
|-----------------------|------|
| 讲义/笔记/考点/教材/备考/教资/科目/地理/数学/语文/英语 | 备考资料 |
| 简历/身份证/证件/学历/学位/毕业 | 个人文档 |
| 配置/config/settings | 配置文件 |
| 备份/backup | 备份 |
| 截图/screenshot/snip | 截图 |
| 项目/plugin/skill/skills | 项目 |
| 日记/反思/复盘/经验 | 个人思考 |

### 归类映射表

| 文件特征 | 归入目标 |
|---------|---------|
| `.md` 参考文档/笔记 | `_tools/参考/` |
| `.pdf` 备考资料 | `个性化/资料/备考/{判断科目}/` |
| `.pdf` 个人证件 | `个性化/资料/个人文档/` |
| `.py` `.js` `.go` `.mjs` 可复用脚本 | `_tools/脚本/{判断功能分类}/` |
| `.zip` `.gz` 压缩包 | `data/backups/` |
| `.png` `.jpg` `.gif` 截图 | `个性化/资料/{判断主题}/截图/` |
| `.json` `.csv` `.sql` 数据文件 | `_tools/脚本/数据/` 或 `个性化/资料/` |
| `_temp/项目/{项目名}/` | `projects/{分类}/{项目名}-{状态}/`（由步骤2规则项扫描触发） |
| `$null` `Thumbs.db` `.DS_Store` | 直接删除 |
| 不确定类型 | `_temp/工作区残留/` |
