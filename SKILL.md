# OpenHanako 优化大师 v2.0

---

## 唯一入口：说「优化大师」

你说「优化大师」，直接展示菜单。

### 菜单

```
⚙ 优化大师 — 请选择：

  说「初始化」       首次使用建立所有目录、规则文档、模板
  说「归类整理」     自动归位工作区文件到对应目录
  说「工作文件夹」   扫描工作区/缓存/会话/临时文件/data
  说「插件技能」     扫描技能/插件/开发规范
  说「Hanako运行」   扫描经验与记忆/配置/运行时缓存

  说「全部」 一次性扫描所有
  说「退出」 结束
```

### 触发

| 你说 | 我做什么 |
|------|---------|
| 「优化大师」 | 展示菜单 |
| 「初始化」 | 建立所有目录/文档/模板 → 回菜单 |
| 「归类整理」 | 扫描工作区根目录 → 按类型匹配目标目录 → 列出确认 → 移动 → 回菜单 |
| 「归类整理 文件路径」 | 对指定文件按规则归类 → 列出建议去向 → 确认 → 移动 → 回菜单 |
| 「工作文件夹」 | 扫工作区+缓存+会话+_temp/+data+_tools/ → 报告 → 确认 → 执行 → 回菜单 |
| 「插件技能」 | 扫技能+插件+开发 → 报告 → 确认 → 执行 → 回菜单 |
| 「Hanako运行」 | 扫经验与记忆+配置+运行时 → 报告 → 确认 → 执行 → 回菜单 |
| 「全部」 | 依次扫三个板块 → 回菜单 |
| 「退出」 | 结束 |

## 归类整理

> 自动识别文件类型并归位到对应目录。说「归类整理」扫全工作区，说「归类整理 路径」指定文件。

### 归类映射表

| 文件特征 | 归入目标 | 示例 |
|---------|---------|------|
| `.md` 参考文档/笔记 | `_tools/参考/` | `笔记排版语法规则.md` |
| `.pdf` 备考资料 | `个性化/资料/备考/{判断科目}/` | `地理讲义.pdf` |
| `.pdf` 个人证件 | `个性化/资料/个人文档/` | `简历.pdf` |
| `.py` `.js` `.go` `.mjs` 可复用脚本 | `_tools/脚本/{判断功能分类}/` | `db_check.py` |
| `.zip` `.gz` 压缩包 | `data/backups/` | `hana-backup-xxx.zip` |
| `.png` `.jpg` `.gif` 截图 | `个性化/资料/{判断主题}/截图/` | `截图.png` |
| `.json` `.csv` `.sql` 数据文件 | `_tools/数据/` 或 `个性化/资料/` | `export.json` |
| `.exe` 可执行文件（有源码） | 提示确认后删 | `app.exe` |
| `$null` `Thumbs.db` `.DS_Store` | 直接删除 | — |
| 不确定类型 | `_temp/工作区残留/` | — |

### 执行流程

```
你说「归类整理」
    ↓
扫描工作区根目录（_temp/_tools/data/projects/个性化/ 之外的所有文件）
    ↓
逐项匹配类型 → 标注建议去向
    ↓
列出待归位文件 → 你逐条确认（做/跳过）
    ↓
执行移动/删除 → 留痕 → 更新索引 → 回菜单
```

你说「归类整理 文件名」只处理指定文件，其余流程同上。

## 初始化流程

> 一次性建立所有规则和框架。之后直接使用三大板块即可。

### 执行内容

```
第1步：建立目录结构
第2步：写入参考文档（3 份）
第3步：写入模板文件（5 份）
第4步：适配检测
第5步：输出报告
```

---

## 三大板块扫描内容

### 工作文件夹
① 工作区（根目录越界/项目待归位/_temp积压）
② 缓存（日志/检查点/图片超量）
⑤ 会话（空目录/大文件）
_temp/ 缓冲区积压
data/ 备份超量/归档可激活
_tools/ 闲置脚本>90天

### 插件技能
③ 技能（同名/废弃/bundle一致/副本）
⑥ 插件（孤儿目录/token-cache/补丁）
⑧ 开发（未结项项目/编译残留）

### Hanako 运行
④ 经验与记忆（pinned超量/经验库合并/可提炼）
⑦ 配置（设置合理性·只读）
⑨ 运行时（session积压/缓存总容量/旧备份）

## 核心执行流程

> 选一个板块，走四步。走完自动回菜单。

### 阶段1：扫描 → 阶段2：报告 → 阶段3：确认执行 → 阶段4：沉淀

操作留痕 `_temp/检查点/` → 自动回到菜单。

## 适配检测

每次扫描前自动检测。未检测到的跳过。

```
有 个性化/资料/备考/           → 备考规则
有 个性化/联动/思源笔记/       → 思源笔记联动
有 个性化/联动/记忆助手-MemOS/ → MemOS 规则
有 个性化/Hanako/             → Hanako 知识库
有 个性化/事项/{事项}/         → 事项三模协作
```
