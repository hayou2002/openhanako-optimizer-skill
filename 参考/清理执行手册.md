# 清理执行手册

> 优化大师参考文档。每项标注适用范围：🟢 通用 / 🟡 个性化（按环境适配）

---

## ① 🟢 系统缓存清理

| 操作 | 规则 |
|------|------|
| 删旧日志 | 保留 auto-update.log + security-audit.jsonl，删3天前.log |
| 删旧检查点 | 保留最新10个 |
| 删图片缓存 | plugin-data/image-gen/ 中7天前 |

## ② 工作区清理

| 操作 | 规则 | 适用 |
|------|------|------|
| 文件归位 | 参考→_tools/参考/；备考→个性化/资料/备考/；压缩包→data/backups/ | 🟢 |
| 项目归位 | `_temp/项目/` → `projects/{分类}/{项目名}-{状态}/` | 🟢 |
| 根目录检查 | 白名单过滤，越界列出建议去向 | 🟢 |
| 残留文件 | `$null`/`Thumbs.db`/`.DS_Store` → 直接删 | 🟢 |
| 截图整理 | 根目录截图/ → 归入对应目录 | 🟢 |
| 临时备份zip | `hana-backup-*.zip` → `data/backups/` | 🟢 |
| _tools/闲置 | 脚本>90天未使用 → 提示归档 | 🟢 |
| data/备份超量 | >7份 → 确认清理 | 🟢 |
| data/归档激活 | 提示是否激活归档项目 | 🟢 |
| 个性化/索引 | 更新各索引.md | 🟡 |

## ③ 🟢 技能清理

| 操作 | 规则 |
|------|------|
| 同名技能 | 列出对比，保留新版 |
| 相似技能 | >70%相似，用户决定 |
| 废弃技能 | default-enabled:false且从未启用 → 确认删 |
| 工作区副本 | 删 workspace/skills/、workspace/skill/ |
| bundle一致性 | 检查 SKILL.md name 与 skill-bundles.json 一致 |

## ④ 经验与记忆清理

| 操作 | 规则 | 适用 |
|------|------|------|
| pinned 去重 | 同主题≥2条合并或删旧 | 🟢 |
| 经验库去重 | 同类>10条合并精简 | 🟢 |
| 经验提炼 | 遍历 experience/ → 写入 Hanako/技能/ | 🟡 |
| 经验库同步 | 检查 experience/ 和 经验库统一格式.md 一致 | 🟢 |
| 认知沉淀 | 检查近期未记录的决策 | 🟡 |
| 辅助提炼 | 扫描标记 🔄 的文件 | 🟡 |
| MemOS 提醒 | 引导 Playground | 🟡 |

## ⑤ 🟢 会话文件清理

| 操作 | 规则 |
|------|------|
| 删空目录 | session-files/ 下空目录全删 |
| 重复文件 | 同名近似文件列出确认 |
| 大文件预警 | >50MB 建议移出 |

## ⑥ 🟢 插件数据清理

| 操作 | 规则 |
|------|------|
| 孤儿目录 | plugin-data/ 下已卸载插件目录 → 确认删 |
| token-tracker 缓存 | >5MB → 确认清空 |
| 各插件数据大小 | >50MB 预警 |
| 补丁兼容性 | 检查 patcher 状态 |

## ⑦ 🟢 运行时缓存清理

| 操作 | 规则 |
|------|------|
| 归档对话附件 | session-files/ 按最后活跃：>14天确认，>30天自动 |
| 旧插件备份 | plugin-backups/ >30天 |
| 缓存总容量 | >500MB 通知 |
| .ephemeral/ | >7天未修改 |
| uploads/temp-upload | 陈旧文件 |

## ⑧ 🟢 _temp/ 缓冲区清理

| 操作 | 规则 |
|------|------|
| 截图超期 | 7天 → 移入 |
| 数据超期 | 14天 → 移入 |
| 脚本超期 | 14天 → 提示可复用→_tools/ |
| 导出超期 | 30天 → 归入 |
| 项目结项 | ≥14天 → 提示移入 projects/ |
| 一次性超期 | ≥7天 → 确认删 |
| 工作区残留 | ≥7天 → 确认删 |
| 检查点超量 | 保留3份 |
