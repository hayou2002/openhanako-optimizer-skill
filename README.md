# OpenHanako 优化大师 v2.1

> Hanako Agent Skill：一站式工作区治理工具

说「优化」建立规则（治本），说「清理」按规则执行清理（治标）。说板块名跑单项。

## 安装

```bash
install_skill --github_url https://github.com/hayou2002/openhanako-optimizer-skill
```

## 使用

| 指令 | 说明 |
|------|------|
| 「优化」 | 建立/更新优化规则约束（治本） |
| 「清理」 | 按规则执行清理（治标） |
| 「初始化」 | 首次使用：建立目录结构、参考文档、适配检测 |
| 「工作文件夹」 | 文件归类 + 扫描缓存/会话/临时文件/data |
| 「插件技能」 | 扫描技能同名/相似/废弃 + 插件审计 |
| 「系统运行」 | 经验记忆合并/提炼 + 配置审计 + 运行时清理 |
| 「全部」 | 一次性扫描所有板块 |

### 交互流程

```
说板块名 → 扫描 → 报告 → 确认执行 → 留痕 → 连贯询问
```

低风险汇总执行，中风险批量确认，高风险逐条确认。随时说「中断」停止。

## 覆盖板块

- 📁 工作文件夹 — 文件归类、缓存（日志/检查点/图片）、会话、_temp/ 积压、data/ 备份、_tools/ 闲置
- 🧩 插件技能 — 技能同名/废弃/bundle一致性、孤儿插件、token-cache、开发残留
- ⚡ 系统运行 — 经验记忆、配置审计、运行时缓存/session/备份

## 文件结构

```
SKILL.md                   核心执行流（v2.1）
参考/
```
