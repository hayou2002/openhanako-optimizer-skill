# OpenHanako 优化大师 v2.0

> Hanako Agent Skill：一站式工作区治理工具

说「优化大师」展示五大菜单，选后扫描确认执行。支持初始化、归类整理、工作文件夹、插件技能、Hanako 运行五大板块。

### 安装

```bash
install_skill --github_url https://github.com/hayou2002/openhanako-optimizer-skill
```

### 使用

| 操作 | 说明 |
|------|------|
| 优化大师 | 展示菜单 |
| 初始化 | 首次使用：建立目录/文档/规则 |
| 归类整理 | 自动归位工作区文件 |
| 工作文件夹 | 扫工作区/缓存/会话 |
| 插件技能 | 扫技能/插件/开发 |
| Hanako运行 | 扫经验/配置/运行时 |

### 文件结构

```
SKILL.md         核心执行流
参考/
├── 优化规则手册.md
├── 清理执行手册.md
└── 文件组织规范.md
```
